O que você vai ver aqui?

Este documento descreve como as informações pessoais são coletadas, usadas e compartilhadas quando você visita ou realiza interações neste Site, que serve como um canal de comunicação para a divulgação do marketing pessoal do titular.Este Site atua como um currículo dinâmico online e conta com serviços de terceiros, como Google™ Analytics para análise de tráfego, Tawk.io para atendimento em tempo real, Jotform para formulário de contato, Raindrop para compartilhar favoritos e Google™ para hospedagem.

Disposições gerais

Esta Política de Privacidade visa esclarecer como é coletado, utilizado, armazenado e protegido cada um dos seus dados pessoais quando você acessa e utiliza este Site. Ela se aplica a todos os usuários do Site, incluindo aqueles que apenas navegam, bem como aqueles que se cadastram para usar os serviços disponíveis.

Dados coletados

Dados de navegação:

  • Endereço IP: usado para identificar a localização geográfica do usuário e melhorar a experiência de navegação.
  • Dados do navegador e do sistema operacional: coletados para otimizar o Site para diferentes dispositivos e navegadores.
  • Páginas visitadas, links clicados e tempo de permanência: utilizados para entender como os usuários interagem com o Site e melhorar seu conteúdo.

Dados de contato:

  • Nome: fornecido quando o usuário preenche o formulário de contato.
  • Endereço de e-mail: coletado para responder ao contato do usuário ou enviar newsletters (opcional).
  • Mensagem: conteúdo da mensagem enviada pelo usuário.

Cookies:

Pequenos arquivos de texto armazenados no dispositivo do usuário para lembrar suas preferências e personalizar sua experiência no Site.

Uso dos dados

Os dados coletados são usados para:

  • Melhorar a experiência de navegação: personalizar o conteúdo do Site, otimizar o desempenho e solucionar problemas técnicos.
  • Fornecer os serviços: responder ao contato do usuário, enviar newsletters (opcional) e gerenciar o formulário de contato.
  • Analisar o tráfego: entender como os usuários interagem com o Site, melhorar seu conteúdo e direcionar campanhas de marketing.

Compartilhamento de dados

Os dados coletados não são vendidos ou compartilhados com terceiros para fins de marketing. No entanto, eles podem ser compartilhados com:

  • Google Analytics: para análise de tráfego.
  • Tawk.io: para atendimento em tempo real.
  • Jotform: para gerenciamento do formulário de contato.
  • Raindrop: para compartilhamento de favoritos.

Armazenamento e segurança dos dados

Os dados coletados são armazenados em servidores seguros do Github™ e da Google™, que implementa medidas de segurança técnicas e administrativas para proteger os dados contra acessos não autorizados, uso indevido, divulgação, alteração ou destruição.

Direitos do usuário

O usuário tem os seguintes direitos em relação aos seus dados pessoais:

  • Acesso: solicitar uma cópia dos seus dados pessoais.
  • Correção: solicitar a correção de dados incorretos ou incompletos.
  • Exclusão: solicitar a exclusão dos seus dados pessoais.
  • Limitação do tratamento: solicitar a suspensão do uso dos seus dados pessoais.
  • Oposição ao tratamento: solicitar que seus dados pessoais não sejam mais usados para fins de marketing.
  • Portabilidade: solicitar que seus dados pessoais sejam transferidos para outro controlador.

Para exercer qualquer um desses direitos, o usuário pode entrar em contato através do formulário de contato disponível no Site.

Alterações na Política de Privacidade

Esta Política de Privacidade pode ser alterada a qualquer momento. As alterações serão publicadas no Site e entrarão em vigor a partir da data de publicação.

Contato

Se você tiver alguma dúvida sobre esta Política de Privacidade, entre em contato através do formulário de contato disponível no Site.
